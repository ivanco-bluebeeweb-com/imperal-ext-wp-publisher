Открывает REST-доступ к SEO-полям Rank Math для приложения WP Publisher.

== Description ==
Служебный плагин-мост. Регистрирует три meta-поля Rank Math
(rank_math_title, rank_math_description, rank_math_focus_keyword)
в WordPress REST API, чтобы внешнее приложение могло заполнять их
при создании черновиков статей. Запись разрешена только пользователям
с правом редактирования записей (edit_posts).

== Installation ==
1. Плагины -> Добавить новый -> Загрузить плагин -> выбрать zip
2. Нажать «Установить», затем «Активировать»
